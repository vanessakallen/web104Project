Firebase Recipe Storage App

Working as a group, build a recipe storage app that stores a recipe for a yummy dish. The application should store each recipe as a document, in side that document will contain the ingredients.

Using Bootstrap

The app should make use of bootstrap. https://getbootstrap.com/docs/3.4/getting-started

Bootstrap has some easy ready made classes for buttons, lightboxes etc. These can be really handy when prototyping web apps to look good. Please use bootstrap for this application.

Working as a group

  1. Discuss how you will use firebase
  2. First create the code that performs the CRUD Operations
  3. Create the GUI for your application using Bootstrap
